# Polymarket R1

Native Polymarket prediction-market viewer built for the Rabbit R1 240×282 screen.

- Live data from Polymarket's free public gamma API
- 6 category tabs: Trending, Politics, Sports, Crypto, Economy, Tech
- Binary and multi-outcome markets with probability bars
- Detail view with all markets, description, volume, and end date
- R1 scroll wheel + touch + PTT support (MLB scroll fix recipe)
- Auto-refresh every 30s (list) / 15s (detail)

## Install
Scan the QR code with your Rabbit R1.
